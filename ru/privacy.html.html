1. Общие положения

Настоящая Политика конфиденциальности устанавливает порядок обработки персональных данных на сайте humantype.sherhn.space и разработана в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных».

Сайт представляет собой бесплатный онлайн-инструмент для замены типографских символов ИИ на человеческие аналоги. Сайт не требует регистрации, не собирает и не хранит пользовательские данные на сервере. Вся обработка текста происходит локально в браузере пользователя.

HumanType работает полностью в браузере. Никакие пользовательские данные не передаются на сервер. Мы не используем файлы cookie, не применяем аналитику и не размещаем рекламу.

3. Категории обрабатываемых данных

Сайт не собирает, не хранит и не обрабатывает какие-либо персональные данные. Текст, вставляемый пользователем в поле ввода, обрабатывается исключительно в браузере с помощью JavaScript и не передаётся на сервер.

4. Цели обработки

Единственная цель функционирования Сайта — предоставление инструмента типографской нормализации текста. Никакие данные не используются в маркетинговых, аналитических или иных целях.

5. Принципы обработки данных

Сайт функционирует на принципах локальной обработки (client-side only). Все вычисления выполняются исключительно в браузере пользователя. Сервер предоставляет только статические файлы (HTML, CSS, JS) и не выполняет серверный код.

6. Права субъекта данных

Поскольку Сайт не обрабатывает персональные данные, права субъекта, предусмотренные главой 3 152-ФЗ, реализуются через отсутствие сбора данных. По любым вопросам обращайтесь по контактам в разделе 9.

7. Защита данных

Передача данных между браузером и серверами GitHub Pages осуществляется по протоколу HTTPS. Сайт не хранит пользовательские данные, что исключает риск их утечки.

8. Изменения политики

Оператор вправе вносить изменения в настоящую Политику. Новая редакция вступает в силу с момента размещения на Сайте.
